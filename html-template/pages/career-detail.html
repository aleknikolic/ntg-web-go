Key Responsibilities

  • Write and optimize complex SQL and scripts

  • Design and develop data pipelines that leverage structured and unstructured data

  • Data ingestion pipelines and ETL processing, including batch and real-time sync

  • Create Views, Functions as per business requirements

  • Write well designed, testable, efficient code that complies with the Project plans and the Industry Standards

  • Create and maintain technical documentation that is required in supporting solutions

Required Qualifications

  • Bachelor’s in computer science

  • 3-4 years of experience

  • 3-4 years of expExperience of working on and optimizing big data pipelines, architectures and data sets

  • Strong analytical skills related to working with un/structured datasets

  • Experience of manipulating, processing and extracting value from different data source

  • Working knowledge of stream processing and highly scalable data stores

  • Experience with big data tools: Hadoop, Spark, Kafka, etc

  • Experience with relational SQL and NoSQL databases, including MySQL, Postgres, MongoDB, Cassandra

  • Good to have experience with data pipeline management tools: Talend, Luigi, Airflow, etc

  • Experience with AWS and/or Azure cloud services

  • Experience with stream-processing systems: Spark-Streaming, etc

  • Experience with object-oriented/object function scripting languages: Python, Java, Scala, etc
